
A responsive temple website built using HTML, CSS, and JavaScript. The website provides information about the temple, its history, religious significance, festivals, galleries, location, and donation details.

## Features

* Responsive design for mobile, tablet, and desktop
* Temple history and introduction
* Festival and event information
* Image gallery
* Location and contact details
* Donation information
* Smooth navigation and user-friendly interface
* Language switcher support

## Technologies Used

* HTML5
* CSS3
* JavaScript
